# Maobook 爪爪日記

### 專案說明

![image](https://i.imgur.com/Bon5q2q.png)  
`資展國際期末小組專題`  
以前後端分離方式使用 React.js 和 Node.js （Express 框架）製作 SPA 動態網頁，撰寫 RESTful API 串接資料庫

### 使用技術

- 前端：HTML5/CSS3、Bootstrap、React.js
- 後端：Node.js、Express.js、MySQL (MariaDB)
- UI/UX：Illustrator、Figma
- 版控：Git / GitHub、nvm
- 套件管理：npm、yarn
